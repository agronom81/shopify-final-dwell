# Copy od Dwell - Shopify Theme

## Educational Purpose

This Shopify theme was created for educational purposes to demonstrate modern e-commerce development practices, theme architecture, and Shopify platform capabilities.

## What is it?

A fully functional Shopify theme showcasing:

-   Responsive design and mobile optimization
-   Product display and cart functionality
-   Multi-language support
-   Accessibility standards (WCAG compliant)
-   Performance optimization techniques
-   Modern JavaScript and CSS practices

## Key Features

-   Product cards with variants and quick add
-   Collection filtering and sorting
-   Cart drawer with recommendations
-   Mega menu navigation
-   Multiple section layouts (hero, slideshow, content blocks)
-   Search with predictive results
-   Gift card support
-   Customizable color schemes and typography

## Tech Stack

-   **Liquid** - Shopify templating language
-   **JavaScript (ES6+)** - Modern frontend functionality
-   **CSS3** - Responsive styling
-   **Shopify CLI** - Development tools
